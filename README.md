# fy-elui
> 一个基于vue-cli3搭建的前端ui框架

## 安装依赖
```
yarn install or npm install
```

## 开启开发环境
```
yarn run serve or npm run serve
```

## 打包构建
```
yarn run build or npm run build
```

## Lints and fixes files
```
yarn run lint
```

## 目录结构
```
|--dist----------------------------------- 打包构建后生成的目录
|--node_modules--------------------------- 安装依赖生成的目录
|--public--------------------------------- 存放静态资源文件和html模板文件
   |--favicon.ico------------------------- 页面标题图标
   |--xxx.html---------------------------- html模板文件
|--src------------------------------------ 开发主目录, 在这个文件夹下开发
   |--assets------------------------------ 系统公共资源文件
   |--components-------------------------- 系统公共组件文件
   |--entrys------------------------------ 入口文件
   |--lang-------------------------------- 多语言包文件
   |--packages---------------------------- fy-ui组件包
   |--styles------------------------------ 系统全局样式文件
   |--utils------------------------------- 工具集文件
   |--views------------------------------- 各子系统模块存放文件
      |--index---------------------------- 首页
      |--login---------------------------- 登录页
|--.browserslistrc------------------------ cli3.0默认生成的
|--.eslintrc.js--------------------------- 使用eslint时, 对代码格式验证配置
|--.gitignore----------------------------- 提交到gitHub仓库时, 过滤文件配置
|--babel.config.js------------------------ babel配置
|--package.json--------------------------- 项目依赖包信息
|--postcss.config.js---------------------- postcss和pxtorem配置
|--vue.config.js-------------------------- 修改内部webpack的配置
|--yarn.lock------------------------------ 使用yarn安装依赖时生成的
|--README.md------------------------------ 说明文件
```
